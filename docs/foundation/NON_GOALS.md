SecretEngine – Non-Goals
Purpose

This document defines what SecretEngine explicitly will not do.

These are permanent unless re-logged.

Engine Non-Goals

No visual scripting system

No general-purpose editor

No blueprint equivalent

No scripting language embedding

No web-based runtime

No dynamic runtime asset import

No full-featured marketplace

No cinematic tools

No engine-as-a-service ambitions

Technical Non-Goals

No OpenGL fallback

No DirectX backend

No automatic multithreading

No reflection-heavy runtime

No magic serialization

No hidden allocations

Business Non-Goals

Not competing feature-for-feature with Unreal

Not selling engine licenses initially

Not supporting every genre

Not chasing trends

Frozen
